# MiniTank Robot - KEYESTUDIO Mini Tank Kit V2

## Overview
A smart tank robot built using the KEYESTUDIO Mini Tank Kit V2 for Arduino. Features include IR remote control, ultrasonic obstacle avoidance, line following, and an 8x16 LED panel display. This project demonstrates **Arduino programming, robotics prototyping, sensor integration, and hardware-software interfacing**.

## Features
- IR Infrared remote control
- Line following and ultrasonic obstacle avoidance
- 8x16 LED panel display
- Smartphone app control (iOS & Android)
- Modular and customizable Arduino code

## Hardware
- KEYESTUDIO Mini Tank Kit V2
- Arduino Uno (or compatible)
- Ultrasonic sensor
- IR receiver & remote
- LED matrix
- Motors & chassis parts (included in kit)

## Code
- Arduino `.ino` files are in `/code/arduino/`
- Modular functions for motor control, sensors, and LED display
- Example scripts in `/examples/`

## Setup & Build
1. Assemble the robot following the included hardware instructions.
2. Install required Arduino libraries (listed in `/code/arduino/libraries/`).
3. Upload `MiniTank_V2.ino` to the Arduino board.
4. Run example scripts to test sensors, motors, and LED display.
